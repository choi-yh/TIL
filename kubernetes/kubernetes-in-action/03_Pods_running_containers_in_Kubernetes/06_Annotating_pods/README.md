# [3.6 파드에 어노테이션 달기](#Index)
* 파드 및 다른 오브젝트는 레이블 외에 annotations 를 가질 수 있다.
* 어노테이션은 key-value 쌍으로 레이블과 거의 비슷하지만 식별 정보를 갖지 않는다.
* 레이블은 오브젝트를 묶을 수 있지만, 어노테이션은 불가능
* 어노테이션은 주로 쿠버네티스에 새로운 기능을 추가 할 때 사용된다.
  * 알바, 베터 버전에는 API 오브젝트에 새로운 필드를 추가하지 않고,
  * 어노테이션을 사용 후 API 변경이 명확해지면 새로운 필드를 추가한다.
  * 새로운 필드가 도입되면 관련 어노테이션 사용이 중단된다.
